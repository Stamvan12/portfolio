GALLERY PHOTOS GUIDE
====================

Place your activity photos in this folder with the following naming:
- 1.jpg
- 2.jpg
- 3.jpg
- 4.jpg
- 5.jpg
- 6.jpg

Recommendations:
- Format: .jpg or .jpeg (PNG also supported but larger file size)
- Size: 1200x800px or similar landscape ratio (4:3 or 16:9)
- File size: Keep under 500KB per image for fast loading
- Content: HR activities like recruitment events, training sessions, team building, office work

The gallery currently displays 6 photos. To add more:
1. Add new images (7.jpg, 8.jpg, etc.)
2. Edit index.html to add more .gallery-item elements
3. Follow the existing structure with data-reveal-delay for staggered animations

Example activities to showcase:
- Conducting interviews or job fairs
- Leading training sessions or workshops
- Team collaboration or meetings
- HR documentation or administrative work
- Employee engagement activities
- Professional development events
